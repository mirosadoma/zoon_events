# Review checklist

- [ ] Tenant scope and cross-tenant denial tested
- [ ] Exact permission/policy and last-admin invariant tested
- [ ] Required audit evidence is atomic, sanitized, and verifiable
- [ ] API/OpenAPI, pagination, idempotency, rate limit, and errors agree
- [ ] Arabic/English, RTL/LTR, accessibility, and dashboard props are safe
- [ ] SaaS/on-premise and offline behavior remain equivalent
- [ ] Documentation, migration, rollback, telemetry, and operational evidence updated
- [ ] No secret, excluded feature, production fake, Docker, or licensed asset
