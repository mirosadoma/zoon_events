# Localization and accessibility

Owner: Admin Console  
Last reviewed: 2026-07-03

English/LTR and Arabic/RTL have equivalent capabilities and validation. Persisted codes
remain language-neutral. Dates and numbers use `Intl`; identifiers/email use bidi-safe
`dir=auto`. WCAG keyboard focus, semantic landmarks, labels, alerts, contrast, reduced
motion, responsive widths, and automated axe/browser checks are release gates.
